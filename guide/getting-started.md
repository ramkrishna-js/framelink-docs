# Getting Started

Framelink is a lightweight, robust, and plugin-ready Lavalink client (v3 & v4) for Node.js. It's designed to be simple to use while providing all the features you need for a modern music bot.

## Why Framelink?

- **Simple API**: Easy to set up and manage.
- **Lavalink v3 & v4**: Full support for both versions.
- **Plugin System**: Extend the library easily.
- **Queue Management**: Built-in, efficient queue system.
- **Type Safety**: Written in TypeScript.
